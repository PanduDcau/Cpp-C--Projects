
# Instructions for Running the Mr. Pacman Game

1. Open the `index.html` file in a web browser.
2. After a 3-second countdown, Pacman will start moving automatically.
3. Use the arrow keys to change Pacman's direction.
4. Eat the dots to increase your score.
5. Avoid hitting the walls. If Pacman hits a wall, the game is over.
6. To restart the game, refresh the web page.

Enjoy the game!
